# KPI 体系：业务与组织（非仅技术指标）

## 1. 分层 KPI（示例结构）

| 层级 | 示例 KPI | 数据源 |
|------|----------|--------|
| **公司** | Runway、月度净收入、现金消耗 | 财务 |
| **增长** | 新客 CAC、邀请 K-factor、渠道 ROI | **`07-metrics-growth`** + 渠道表 |
| **交易** | GMV、入金率、完成率、争议率 | 链上事件 + DB 投影 `spec/01` |
| **供给** | 活跃向导、审核通过率、接单时延 | 运营 + API |
| **技术** | 可用性、P0 事故、索引延迟 | SRE、`spec/120` |

## 2. 与技术指标的边界

- **「索引器落后 X 分钟」** 是 **SRE KPI**，**不能**单独证明商业成功。  
- 董事会级应至少看到 **GMV / 收入 / 供给密度** 之一。
