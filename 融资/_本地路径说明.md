# 本地路径说明

骨架里的链接默认使用：

```text
../TT-Expedition-docs/
```

表示：**本 `融资` 文件夹的上一级目录下的 `TT-Expedition-docs` 文件夹**（即与 `融资` 并列放在桌面时的常见布局）。

若你的主仓库在其他盘符或目录，请任选其一：

1. 在资源管理器里把 **`融资`** 夹挪到与 **`TT-Expedition-docs`** 并列的位置；或  
2. 用编辑器在 `融资` 目录内 **全局替换** `../TT-Expedition-docs/` 为你的实际路径（建议仍用相对路径，便于团队统一）。

**不要**把含真实用户数据、密钥、律所保密附件的路径写进公开 Git 仓库正文；可写「Off-repo：网盘路径 + Owner」。
