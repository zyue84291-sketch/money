# 00 — Live Product Proof（线上可验证存在）

**用途**：Partner / IC 可在 **5 分钟内** 自助验证「不是 PPT」。**禁止** 仅写「即将上线」而无可点链接（除非书面标注 *Staging + 访问方式 + 过期日*）。

## 2. 最小可验包（Cold IC 场景）

- **1 条主路径录屏**（注册 → 下单/托管 → 结算或争议入口）+ **同一路径** 的 **1 笔测试网/主网 Tx 哈希**（若适用）。  
- **1 张架构图**（与 **[`07`](../07-技术尽调-骨架.md)** 一致版本号）。  
- **Uptime / 错误率**（若有）：最近 30d 摘要或「尚未生产级」书面声明。

## 3. 门禁（与 [`09`](../09-材料门禁与核对清单-骨架.md) 一致）

- [ ] 表中 **每一格 URL** 由 **非作者** 在 **48h 内** 二次点开  
- [ ] 无公开生产环境时，**Staging** 须有 **过期日与替代 Demo**  
- [ ] 含 **用户数据** 的录屏 / Dashboard **已脱敏**
