### 牵引指标（Traction Anchor）— 定义同窗

| 指标 | 登记表 / 附件 | 证据类型（DD） |
|------|----------------|----------------|
| MAU（旅行者） | 登记表 §0 + 指标字典（VDR User Metrics） | BI 导出 / 埋点变更记录 |
| Escrow 活跃用户 | 登记表 §0 | 链上事件投影 + 内部对账表 |

| Commit→Escrow 等主转化率 | 登记表漏斗段 | 漏斗定义与 SQL 版本戳 |
