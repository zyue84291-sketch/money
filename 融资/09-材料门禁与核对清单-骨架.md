## 对外前勾选（最小集）

- [ ] **`spec/08-4`** 与 Deck 中 **所有金额/比例/状态表述** 已逐条对照  
- [ ] **`11/06`**（或当期等价表）**A～F** 与本轮材料 **一致**  
- [ ] **`22/07`**（经营真值 / Board 口径）与材料 **无打架**  
- [ ] **`99`** / **`evidence/`** 中宣称的验证 **有链或书面阻塞**  
- [ ] **未在 `evidence` 或 `22/07` 登记** 的结论 **未写入** `11/06` 资本叙事可引用层

## 企业级填充（PM · 门禁 = 防止「一次失误毁掉融资」）

### 对外发包分级

| 级别 | 包含物 | 最低门禁 |
|------|--------|----------|
| **L0 公开** | 官网、社媒 | `08-4` 单条审核 |
| **L1 投资人** | Deck、白皮书摘要、月报 | 本节最小集 + **`11/06`** |
| **L2 DD** | Data Room 全量 | **`01-Data-Room-索引`** + **`14`** 机构树 + 法务 |
| **L3 Board** | Board pack | **`33-board-governance`** + CFO |

### 48h 前强制四连（建议写进日历）

1. **`11/06`** 全表（A～F）  
2. **`08-operations-pm/audit/01`** §2 数字一致性  
3. **`05-go-to-market/00`** 材料顺序与 **治理联动脚本**（若触达 82～84）  
4. **`99/06` Day15** 或等价「证据就绪」勾表  

### 常见失败模式（当场拒发）

| 模式 | 纠正动作 |
|------|----------|
| Deck 数字与 `22` 周会不一致 | 以 **`09/00` §0** 重刷全材料 |
| Token 页与 `LEGAL-SIGNOFF` 不一致 | 删页或更新 **`spec/82`～`84`** 后重发 |
| 无第二城证据却写「全国复制」 | 收窄为 **`23/04`** 已达成的条目 |
| 律所意见书未闭合却写「已背书」 | 改文案为「内部讨论稿」 |
