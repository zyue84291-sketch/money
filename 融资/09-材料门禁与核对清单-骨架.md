## 企业级填充（门禁 = 防止「一次失误毁掉融资」）

### 对外发包分级

| 级别 | 包含物 | 最低门禁 |
|------|--------|----------|
| **L0 公开** | 官网、社媒 | `spec/08-4` 单条审核 |
| **L1 投资人** | Deck、白皮书摘要、月报 | 本节最小集 + **本文件数字对齐表** |
| **L2 DD** | Data Room 全量 | **VDR 主索引** + [`14`](./14-机构级-Data-Room-Index-骨架.md) + 法务 |
| **L3 Board** | Board pack | [`17-投后治理-Post-Investment-Governance-骨架.md`](./17-投后治理-Post-Investment-Governance-骨架.md) + CFO |

### 48h 前强制四连（建议写进日历）

1. **数字对齐表** 全表（A～F）  
2. **跨材料数字一致性**（Deck / `02` / 月报）  
3. **材料顺序** 与 **治理 Token 披露**（若触达 `spec/82`～`84`）经法务  
4. **[`08`](./08-试点与证据-Data-Room-骨架.md)** 或等价「证据就绪」勾表  

### 常见失败模式（当场拒发）

| 模式 | 纠正动作 |
|------|----------|
| Deck 数字与周会不一致 | 以 **[`02`](./02-经营真值-指标与财务-骨架.md) 登记表 §0** 重刷全材料 |
| Token 页与 `LEGAL-SIGNOFF` 不一致 | 删页或更新 **`spec/82`～`84`** 后重发 |
| 无第二城证据却写「全国复制」 | 收窄为 **[`08`](./08-试点与证据-Data-Room-骨架.md)** 已达成的条目 |
| 律所意见书未闭合却写「已背书」 | 改文案为「内部讨论稿」 |
