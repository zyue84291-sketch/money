## 1. IPO 可能性（若适用）

**要写清**：**可想象的最小实体形态**、**收入规模与合规门槛**、**时间数量级假设**（非承诺）。

- [ ] 与 **当前公司架构**（`13`）**不矛盾**  
- [ ] 对外口径为 **「optionality」** 而非 **承诺时间表**  

---

## 2. Token 流动性（若适用）

**要写清**：**上市所 / DEX** 策略、**锁仓与流通量**、**做市与合规**（机构常问）。

- [ ] **不与证券法冲突** 的表述已通过法务  
- [ ] **内部人与投资人解锁曲线** 与公开材料一致  

---

## 3. 并购路径（M&A）

**要写清**：潜在 **战略买家类型**（OTA、支付、链上基础设施）、**协同叙事**、**历史上可比交易**（仅作区间参考）。

- [ ] 与 **`11`** 中战略基金/产业资本 **联动**（谁既是买家又是渠道）  
- [ ] **控制权溢价** 与 **少数股权保护条款** 在 `10`/`12` 中有意识  

---

## 4. 二级市场策略（股权 / 老股）

**要写清**：是否考虑 **secondary**、**老股转让**、员工期权流动性；**与本轮发行的关系**。

- [ ] **创始人卖老股** vs **公司发新股** 对叙事与士气的 **内部共识**  
- [ ] **Right of first refusal** 等条款在 cap table 已考虑  

---

## 5. 对不同类型投资人的一句话

| 类型 | 退出叙事侧重点 |
|------|----------------|
| Web3 基金 | Token 流动性 + 协议收入 + 并购中的协议价值 |
| 传统 VC | M&A + 长期 IPO optionality + 收入倍数路径 |
| 家办 | 下行保护、分红/回购（若业务允许）、条款 |

- [ ] **与 `26` 文件** 无 **互相矛盾的时间或倍数假设**
