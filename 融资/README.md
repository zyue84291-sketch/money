# 融资材料包（投资人 / DD 用）

本目录为 **面向一级市场与机构尽调的自包含材料包**：叙事、门禁、Data Room 索引与白皮书均以 **本文件夹内 Markdown** 为对外主入口。**不向投资人展示** 内部产品幕僚仓库路径；工程与协议真值以 **Data Room（VDR）中已归档的技术与法律附件** 及 **`07-技术尽调`** 所引用的规格为准。**禁止** 在本包正文手写与 **`02-经营真值`** 中「对外牵引数字登记表」§0 已双签数字 **相冲突** 的口径。
