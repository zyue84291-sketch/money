## PM 建议的可选扩展（未单独成文件时可并入 17/18/20）

- **银行与资金路径 / 对账**：详细落 **`18`** §2；`14` Finance 只放索引。  
- **公关与危机沟通**：战时 playbook 见 **`41`**；**投资人通知顺序** 与 **`17`** §4 对齐。  
- **工程级安全体系**：流程与 runbook 见 **`38`**；审计报告索引见 **`20`**。  
- **Bridge / 可转债应急**：下一轮融资延迟时的 **条款红线**（与 `10` 联写）。  
- **Side letter 与 MFN 索引**：所有「非标准条款」**单列一页**，避免 DD 漏披露。  
- **Advisor / 生态换股**：cap table 与叙事 **一致**（并入 `13`）。  
- **Co-sell**：与 **`25`** 战略伙伴的 **联合打单** 计划（并入 `11`/`12` 优先级）。  
- **D&O / 网络安全保险**：保单摘要与续期（常进 **`20`** 或 `14` Legal，机构偶尔会问）。
