# 融资材料包（本地骨架）

本目录是 **给投资人 / DD 用的材料包占位**：正文与真值仍以主仓库 **`TT-Expedition-docs`** 里的 `product-manager/`、`spec/`、`product-management-office/` 为 **SSOT**；此处 **不复制** 大段正文，只放 **清单、核对项、摘要占位** 与 **路径指针**。

## 主仓库根目录（请按你本机实际修改）

默认按「融资夹与主仓库同在桌面」书写相对路径：

| 说明 | 相对路径（从本 `README.md` 所在目录出发） |
|------|---------------------------------------------|
| 主仓库根 | `../TT-Expedition-docs/` |
| PM Office | `../TT-Expedition-docs/product-management-office/` |
| 执行稿（Pitch 等） | `../TT-Expedition-docs/product-manager/` |
| 工程与协议 SSOT | `../TT-Expedition-docs/spec/` |

若你的主仓库不在桌面，请全文替换前缀或见 **`_本地路径说明.md`**。
