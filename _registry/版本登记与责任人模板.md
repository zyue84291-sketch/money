# 文档变更登记（模板）

复制本表头后填写；**重大对外口径变更**须同步法务与 `spec/08-4`。

| 字段 | 内容 |
|------|------|
| **文档路径** | `product-management-office/...` |
| **变更类型** | 新增 / 修订 / 废止 |
| **版本** | 例：0.1.0 → 0.2.0 |
| **日期** | YYYY-MM-DD |
| **作者** | 姓名 |
| **评审** | 姓名（产品 / 工程 / 法务 勾选） |
| **摘要** | 一句话 |
| **关联 spec** | 例：`spec/01` §〇 |
| **对外影响** | 无 / 有（若有则附法务工单或 08-4 引用） |
