# 推荐与裂变机制

## 1. 设计要素（ checklist ）

| 要素 | 问题 |
|------|------|
| **动机** | 邀请者 / 被邀请者各得什么？是否双边奖励？ |
| **归因** | 邀请码、链接、链上 referral 字段？与 **`x-request-id` / 埋点** 一致？ |
| **风控** | 女巫、批量注册、自邀自；与 **`spec/03`**、rate limit 对齐 |
| **合规** | 是否构成证券或拉人头；**法务** |
| **工程** | 若新增 API → **`spec/04` §3.4** 先行 |

## 2. 与 Web3 特有增长
