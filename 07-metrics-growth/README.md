# 07-metrics-growth — 指标与增长

**目标**：**北极星、过程指标、埋点口径** 与 **`spec/04`**、现有 PM 表对齐。

**与 `10-growth-engine/` 的分工**：本目录 = **「量什么、怎么复盘」**；**`10-growth-engine/`** = **「用什么机制从 0 滚到规模化」**。二者须闭环：每个增长机制至少挂一组 **可证伪指标**。

**与 `15-data-decision/` 的分工**：**`15`** = **北极星、漏斗阈值、A/B、实验排期**（用数据做决策）；本目录 = **埋点字典与口径、复盘节奏**；会议节奏见 **`15/04`**。

**与 `22-executive-dashboard/` 的分工**：**`22`** = **CEO/VC 层唯一北极星、KPI 树、经营复盘一页纸**；**`22/04`** = **各支柱与 NS 子指标绑定表**；本目录 = **指标定义与数据源对齐 `spec/04`**；二者口径须 **同一链接**，避免两套数。
