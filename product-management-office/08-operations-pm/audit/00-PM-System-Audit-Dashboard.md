# 00 — PM System Audit Dashboard（系统级审计层）

**用途**：用 **一张检查面** 回答管理层与外部 DD：**「这套 PM 体系是否在运转？关键数字能否入账？出事能否回滚？谁真正能决策？」** —— **不是**替代 **`spec/15` 附录〇`** 或 **工程 CI**，而是 **组织与商业可信度** 的 **总览**。

**频率**：**月** 全量；**季** 签字归档（建议 **CPO + CFO** 或章程约定）；**融资 / 监管窗口前** **强制** 跑一次。

**Owner**：**PM Owner**（无则产品负责人）维护 **证据链接**；**CFO** 对 **M2/M3** 子项有 **否决未通过则不得对外报收入类数字** 的业务规则（见 **`07/06`**）。

**总判定**：

- **任意 M1～M9** **红** → **系统不可宣称「企业级就绪」**；**融资 / 监管材料** 须 **降级表述** 或 **停发**。**M10** 为 **趋势辅助**；**M8/M9 红** 时 **M10 不得单独宣称「健康」**。  
- **无红、黄 ≤ ____ 项（自填）** → **可融资工作态**（仍须法务 **`spec/08-4`** 终审）。
