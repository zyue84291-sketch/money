## 2. BD Pipeline（**线索 → 签约 → 上线**）

| **阶段** | **定义** | **出口标准** | **系统字段（建议）** |
|----------|----------|--------------|----------------------|
| **Lead** | 首次会议 / 展会名片 | 记 **来源渠道** + **伙伴类型** | `partner_id` |
| **Qualified** | 双方确认 **合作类型**（OTA/Web3/支付…） | 对齐 **`25/05`** 一行单位经济 **草稿** | `stage_target` |
| **Pilot（S1）** | 试点合同签署 | **`spec/04`** 集成项 **清单非空** | `s1_start` |
| **Revenue（S2）** | 归因 GMV 发生 | **`25/06`** S1→S2 条件 **打勾** | `s2_start` |
| **Strategic（S3）** | 深度/API/排他（若适用） | **法务索引 + `26/05` 集中度不爆** | `s3_start` |
| **Churned** | 停止合作 | **原因码** + 复盘进 **`21/07`** | `closed_reason` |

## 3. **每周** 最低动作（**BD + PM-O 联席**）

| **天** | **动作** | **产出物** |
|--------|----------|------------|
| **周一** | Pipeline 清点：新增 Lead、停滞 **>14d** 标红 | 表更新（可 Confluence/Sheet） |
| **周三** | **S1→S2** 候选：过 **`25/06`** 门槛 + **`22/07`** 是否可填归因 GMV | **Go / No-Go** 一行 |
| **周五** | 下周 **3 个** 高优先级触达 + **1 个** 工程/法务 unblock | 写入 **`06`** 今日任务或周纪要 |

---

## 4. KPI（**周 / 月**）

| **KPI** | **定义** | **健康区间（示意）** |
|---------|----------|----------------------|
| **Pipeline 价值** | Σ（加权 **预期年化贡献毛利**） | 月环比 **不下降**（早期允许波动） |
| **S1→S2 转化率** | 本周期达标 / 本周期结束试点 | **≥ 40%**（调参见 **`25/06`**） |
| **S2→S3 推进中** | 合同在法务 **≤45d** | 超期 **升板** |
| **上线延迟** | Pilot 签约 → 生产可用 | **P90 < 30d**（视集成复杂度） |
| **伙伴相关争议率** | 伙伴归因订单争议 / GMV | **< `25/06`** 全局阈值 |
