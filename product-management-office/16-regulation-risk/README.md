## 合规贯穿机制（Discovery → Definition → Go-to-market）

**任何** 经 **`03-discovery/` → `04-definition/` → `05-go-to-market/`** 进入 **对外或上线叙事** 的需求 / Epic **必须**：

1. **标注 `spec/08-*` 引用**（在 PRD **`01` 附录 A** 或 **`_registry/新需求-最小入口-模板`** 可审计位置）；  
2. **通过 `05-go-to-market/00-对外叙事与材料门禁-082-084-08-4.md`**（或你们等价 **法务门禁工单**）；  
3. **仍须满足** **`01` SSOT** 发版与 **`06-delivery-release/`** 工程门禁。

**否则**：**不得** 用于 **对外材料 / 融资路演 / 公开发布**；**上线宣称** 按 **`01` §2.5** 视为 **无效上线风险**。

**合规漂移**：**收费 / 国家 / 资金路径** 等变更后的 **复审与文档回写** 见 **`00` §6（Drift Control）**。

| **域** | **本目录已覆盖（可指给 DD）** | **仍须组织填实（外部交付物）** |
|--------|-------------------------------|----------------------------------|
