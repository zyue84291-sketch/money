## 2. First 7 Days Checklist（交接包 · 勾选）

### Day 1

- [ ] 获得 **`_registry/权限边界-强执行.md`** 中列出的 **仓库 / 看板 / 财务只读** 权限

### Day 2

- [ ] 跟 **周会** 一次（**`meetings/01`** + **`22/06`**），只观察 **不抢话**  
- [ ] 跑通 **`spec/04`** 与 **`07/00`** 中 **NSM 相关** 的 **一条** API/埋点路径（只读）

### Day 3

- [ ] 与 **Finance** 对齐：**`09/07`** + **`07/06`** 上月结论 **1 页**  
- [ ] 读 **`06/02`** 资金断流顺序（桌面推演可旁听）

### Day 4

- [ ] **`11/06`** 表走读（不要求全勾，**标黄项**）  
- [ ] 与 **法务** 15min：**`08-4`** 当前禁区

### Day 5

- [ ] 在 **`meetings/`** 起草 **一份** 纪要（试用 **`meetings/02`** 质量闸）  
- [ ] 更新 **`Office运行健康度`** 一行（若当周值周）

### Day 6

- [ ] **`audit/00`** 月检 **模拟填半表**（与卸任对答案）  
- [ ] 与 **工程接口人** 确认 **下一发布窗** 与 **`06/00`** 状态

- [ ] **书面**：**「可独立主持周会 + 可定位 SSOT」** 自评 + **卸任签字**  
- [ ] 若未通过 → **延长** 3 天并 **记入 `21/07`**
