# 优先级体系（P0 / P1 / P2）

## 1. 定义（产品语言）

| 级别 | 含义 | 典型后果 |
|------|------|----------|
| **P0** | 不做则 **资损、违法、主链不可用** | 阻塞发版；见 **`spec/01` §10**、**`spec/15` 附录〇** |
| **P1** | 不做则 **增长/转化显著受损** | 排入当前或下一 Release Train |
| **P2** | 体验优化、长尾 | 资源允许时插入 |

## 2. 与工程 P0 的关系

- **工程 P0**（12 缝、17 条等）以 **`spec/01` §10** 为准；**产品 P0** 不得稀释工程 P0 资源 **无评审**。  
- **合并评审**：双周 **`20-product-iteration`** 会议或 **`08-operations-pm/meetings/`**。

## 3. 升级规则（事件驱动 · **与工程 P0 合并评审**）

| 条件 | 动作 |
|------|------|
| 线上资金类缺陷 | 自动 **P0** + 触发 **`19-risk-operations`** |
| 漏斗跌破 **`15`** 红线 | 见 **§4** **数据触发**；默认 **P0** 若命中 **§4 第一表** |

---

## 4. **P0 / P1 / P2 量化触发**（**减少「语义吵架」**）

> **原则**：**能读数则按表升降**；表外争议 → **`21/02`** + 纪要。**指标定义** 以 **`07/00`**、**漏斗红线** 以 **`15/01` §2`** 为准。

### 4.1 **自动 P0**（**任一成立即 P0**，**24h 内** 进入排期顶栏）

| **信号（默认口径）** | **阈值** | **指标键（`07/00`）** | **动作** |
|----------------------|----------|-------------------------|----------|
| **Commit → Escrow 转化** | **< 60%**（7 日滚动） | `escrow_conversion_rate` | **P0** + 执行 **`15/04` §2.1**（停投放 / **EXP-0142** / Owner） |
| **旅行者 DAU** | **周环比 ≤ −20%** | `dau_traveler` | **P0** + **增长 + 数据** 联席根因 |
| **支付上下文 RPC 错误率** | **> 5%**（7 日滚动） | `rpc_error_rate_checkout` | **P0** + **工程 On-call** + **暂停** 相关实验直至解除 |
| **线上资金 / 合规阻断** | 定性 | — | **P0**（**`spec/01` §10**、**`16`**） |
| **法律背书过期 / 未签发却对外使用** | **`16/05`** 复审日到期 **或** 材料引用 **未签字 Memo** | — | **P0** + **停对外材料刷新**（**`05/00`**） |
| **Penta / 会计落表未闭合却宣称闭环** | **`16/04` §2.3** 缺任意一方 **或** **`16/02` §5** CFO 未勾选 | — | **P0**（**融资/路演门**） |

### 4.2 **自动 P1**（**高优但不挤占 §4.1 队首**）

| 信号 | 阈值 | 说明 |
|------|------|------|
| `escrow_conversion_rate` | **60%～红线目标之间** 且 **连续 5 日下滑** | 优先排 **钱包/撮合** Story，**未**达 §4.1 |
| `weekly_escrowed_gmv`（NSM） | **周环比 ≤ −10%** | **周会第二屏** 已定义；排 **L1** 诊断 Sprint |

### 4.3 **P2**

- **未触发 §4.1～4.2**、且 **与当季 Sprint Goal 无映射**（**`20/05` §2**）→ **默认 P2** 或 **Backlog**。

**与 NS**：凡 **§4.1 P0** 须 **`22/04`** 填 **受影响 `ns_leaf`**（与 **`21/04`** 一致）。
