# 05 — 拟真实验全链路（纸面可跑通 ×2）

**用途**：证明 **H → C → 指标 → ICE → 样本量 → 结果 → 产品决策** 在 **`15/05`** 六阶段下 **无需代码也可推演完**。与 **`06-实验台账与复盘模板`** 字段一一对应。

---

## 实验 1：减少钱包教育步骤 → 提高 Escrow 转化

| 阶段 | 内容 |
|------|------|
| **H** | 减少一步 `wallet_explain` → **Commit→Escrow** 转化率 **+10%**（相对） |
| **C** | 对照组保留 3 步教育；实验组 **折叠第 2 步**（仅文案 +「展开」） |
| **指标** | `escrow_conversion_rate` = `escrowed_users` / `commit_users`（同窗 **7 日滚动**） |
| **事件（占位 → 对齐 `spec/04`）** | `commit_order`；`tx_success` 或 `order_escrowed`（与链上 `Deposit` 投影一致） |
| **来源** | **DB**（会话）+ **on-chain event**（`spec/110` 索引） |
| **ICE** | Impact **8**；Confidence **6**；Ease **7** → **21** |
| **最小样本（写死）** | 每臂 **≥ 1000** **sessions**（主指标若以 **user** 计则 **≥ 1000 commit_users**，工单 **二选一**）；不足 → **不判赢、只延长** |
| **显著性（写死）** | 主指标 uplift **p < 0.05**（双尾）才允许 **全量** |
| **保护机制（写死）** | **核心转化** `escrow_conversion_rate` 对照臂 **相对 ↓ > 5%**（同 7 日窗）→ **立即停止** 分流 + **回滚**（**`15/02` §4**） |
| **预期** | 点估计 **+10%**；**MDE 讨论值 +5%** |
| **结果决策** | **> +5%** 且 **p < 0.05** 且 **护栏未触发** → **全量**；相对 uplift **&lt; 0** → **回滚**；**0～5%** → **50%** 再观察 1 周 |
| **风险** | 误触增加 → 监控 `rpc_error_rate_checkout`（**`07/00`**） |
| **台账 ID** | `EXP-2026-0142` |

## 实验 2：Escrow 页「资金托管安全说明」→ 信任 → Commit 提升

| 阶段 | 内容 |
|------|------|
| **H** | 展示 **资金流一页图**（对齐 **`spec/13`** 资金区克制）→ **Intent→Commit** **+8%** |
| **C** | 对照组无模块；实验组 Escrow 结账页 **折叠卡片「资金如何托管」**（默认收起） |
| **指标** | `intent_to_commit_rate`；辅助 **`trust_score_proxy` B**（**`07/04`**） |
| **事件（占位）** | `intent_session_start`；`commit_order`；`trust_fund_flow_expand` |
| **来源** | **DB** |
| **ICE** | Impact **7**；Confidence **5**；Ease **6** → **18** |
| **最小样本** | 每臂 **≥ 1000** **sessions**（与 **`15/02` §4** 一致） |
| **显著性** | **p < 0.05** 才全量 |
| **保护机制** | **`intent_to_commit_rate` 对照 ↓ > 5%**（同窗）→ **急停** |
| **风险** | **信息过载** → Commit **下降**；若 **&lt; -2%** → **立即停** |
| **结果决策** | **+5%** 且 **p < 0.05** 且护栏未触发 → 全量并 **回写 `spec/13-1`**；否则弃 |
| **台账 ID** | `EXP-2026-0143` |
