# 指标字典与埋点（对齐原则）

## SSOT

- **HTTP 与埋点落点**：以 **`../spec/04-后端与API.md` §3.4** 为准；前端埋点示例参考仓库 `frontend/lib/analytics.ts`（见 **`../spec/00-文档索引.md`** 对 04 的说明）。  
- **产品指标表（执行）**：`../product-manager/31-产品指标与埋点口径表.md`。

## 命名原则

- 指标 Key **稳定**；变更须版本化并在 **`../product-manager/31`** 登记。  
- 与 **资金状态** 相关的展示指标须标注数据源：**链上事件 / DB 投影**，禁止混名（对齐 **`../spec/14`** B110-SSOT-07 精神）。

## 指标字典（**可计算 · ≥24 条** · 事件名 **占位** → 回填 `spec/04` §3.4）

> **每条必须能进实验或进阈值**：`metric_id` 为 **snake_case**；**事件** 须最终与 **HTTP 埋点 / 后端审计** 同名登记。

| `metric_id` | **公式** | **事件（占位）** | **来源** | **漏斗归属** | **决策用途（示例）** |
|---------------|----------|------------------|----------|----------------|----------------------|
| `visit_to_account_rate` | `new_accounts` / `visits` | `page_view_landing`；`account_created` | DB | `funnel_visit_to_account` | **<25%** → 停投放（**`15/01`**） |
| `account_to_intent_rate` | `intent_sessions` / `active_accounts` | `intent_session_start`；`account_login` | DB | `funnel_account_to_intent` | **<40%** → 空状态实验 |
| `intent_to_commit_rate` | `commits` / `intent_sessions` | `commit_order`；`intent_session_start` | DB | `funnel_intent_to_commit` | **<30%** → 撮合 UX |
| `escrow_conversion_rate` | `escrowed_users` / `commit_users` | `commit_order`；`tx_success` / `order_escrowed` | **DB + on-chain** | `funnel_commit_to_escrow` | **<60%** → 停投放 + 实验 **0142** |
| `escrowed_to_completed_rate` | `completed_orders` / `escrowed_orders` | `order_completed`；`order_escrowed` | DB + on-chain | `funnel_escrow_to_complete` | **<65%** → 供给 SLA |
| `weekly_escrowed_gmv` | **SUM** `escrow_amount_usd`（周窗 Escrowed） | `order_escrowed`（金额字段 TBD） | **on-chain + DB** | **NSM** | **周会第一数字**（**`15/00`**） |
| `trust_score_proxy_a` | `escrowed_sessions` / `payment_page_sessions` | `checkout_payment_view`；`order_escrowed` | DB + on-chain | `funnel_commit_to_escrow` | 信任设计 A/B（**`07/04`**） |
| `trust_score_proxy_b` | `commits_after_trust_expand` / `trust_fund_flow_expand` | `trust_fund_flow_expand`；`commit_order` | DB | `funnel_intent_to_commit` | 文案实验 **0143** |
| `time_to_first_escrow_hours` | `median(first_escrow_at − account_created_at)` | `account_created`；`order_escrowed` | DB | onboarding | **↑** 魔法时刻 |
| `wallet_connect_success_rate` | `wallet_connected` / `wallet_connect_prompt` | `wallet_connect_prompt`；`wallet_connected` | DB | `funnel_commit_to_escrow` | 低 → RPC/钱包排查 |
| `approve_tx_success_rate` | `approve_ok` / `approve_submit` | `approve_submit`；`approve_ok` | DB + on-chain | `funnel_commit_to_escrow` | 低 → gas/网络文案 |
| `deposit_tx_success_rate` | `deposit_ok` / `deposit_submit` | `deposit_submit`；`deposit_ok` | DB + on-chain | `funnel_commit_to_escrow` | 低 → txMachine |
| `checkout_drop_rate` | `1 − escrowed_sessions/checkout_started` | `checkout_started`；`order_escrowed` | DB | `funnel_commit_to_escrow` | 定位支付页流失 |
| `rpc_error_rate_checkout` | `rpc_errors` / `rpc_calls_checkout` | `rpc_error`；`rpc_call`（属性 `context=checkout`） | 日志 | health | 与 **0142** 风险联动 |
| `dau_traveler` | `COUNT DISTINCT traveler_id`（日活） | `app_heartbeat` / `page_view`（TBD） | DB | input | 与投放联动 |
| `paid_users_daily` | `COUNT DISTINCT user_id`（当日 Escrowed） | `order_escrowed` | on-chain + DB | output | NSM 辅助 |
| `refund_request_rate` | `refund_requests` / `completed_orders` | `refund_open`；`order_completed` | DB | health | 高 → 信任/质量 |
| `dispute_open_rate` | `disputes_opened` / `escrowed_orders` | `dispute_open`；`order_escrowed` | DB | health | 链 **`spec/03`** |
| `cac_blended_30d` | `spend_30d` / `new_paid_users_30d` | 财务 spend；`account_created` | 财务 + DB | input | LTV/CAC（**`09/02`**） |
| `ltv_proxy_90d_revenue` | `sum_net_rev_90d` / cohort_users | `revenue_recognized`（TBD） | 财务 + DB | output | 单位经济 |
| `experiment_split_balance_ratio` | `min_arm_n` / `max_arm_n` | 实验分流日志 | DB | meta | **>0.95** 才解读 |
| `intent_sessions_per_account` | `intent_sessions` / `active_accounts` | `intent_session_start` | DB | `funnel_account_to_intent` | 需求深度 |
| `market_listing_ctr` | `listing_clicks` / `impressions` | `market_impression`；`listing_click` | DB | visit | 供给发现 |
| `guide_accept_latency_p50` | `p50(accept_at − commit_at)` | `guide_accept`；`commit_order` | DB | `funnel_intent_to_commit` | 撮合延迟 |
| `indexer_lag_blocks` | `head − indexed`（TBD） | indexer health | 运维 | health | 影响 Paid 展示真值 |

**维护**：Owner = **Data**（定义）+ **Finance**（金额类）；**月** 对 **`product-manager/31`** diff 一次。

**财务对账与 ROI**：金额类指标 **入账 / 披露** 须与本目录 **`06-财务对账闭环-链上GMV与收入口径.md`**、**`07-预算到回报-增长ROI与回本模型.md`** 同周对拍；`metric_id` 上增加 **`reconciliation_tier`**（`product_only` / `management_adjusted` / `gaap_ready`）由 **Finance** 标定。
