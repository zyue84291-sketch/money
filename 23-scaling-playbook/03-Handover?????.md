# 03 — Handover、培训与验收

**用途**：规模化时 **人换城不换质**；验收标准与 **`17`** 里程碑对齐。

---

## 1. Handover 最低交付物

| 交付物 | 验收人 |
|--------|--------|
| 更新后的 **城市启动包**（`23/01` 实例） | 区域负责人 |
| **指标看板** 权限与口径说明 | 数据 + 产品 |
| **客服与风险 SOP** 链接与演练记录 | 运营 + **`19`** owner |
| **30/60/90 天目标** | 产品负责人 |

## 2. 培训清单（建议）

1. 产品与资金路径（**Happy path** 一句话 + 争议路径）  
2. 合规禁区与对外话术（**`16`**）  
3. 工单与升级（**`19`**）  
4. 周会节奏（**`22/03`**）
