# 06 — 实验台账与复盘模板

**用途**：**可审计** 的实验流水；VC / 内部问「你们怎么证明在学」时 **直接导出**。

---

## 1. 台账字段（最低）

| 字段 | 说明 |
|------|------|
| `exp_id` | 唯一 ID |
| `hypothesis` | 一句话 |
| `ns_leaf` | 绑定的 **`22/04`** 子指标代码 |
| `start` / `end` | 日期 |
| `status` | running / shipped / stopped |
| `result` | 显著性结论 + 业务决策 |
| `links` | 看板 URL、PR、**`20` 纪要** |

---

## 2. 复盘会模板（30 min）

1. 本周 **结题** 实验（5 min）  
2. **失败** 价值（10 min）— 必须写「不学什么」  
3. 对 **`22` NS** 的净影响估计（10 min）  
4. 下周 **新开** 实验上限 **N 个**（5 min）— 防并行爆炸
