# TAM / SAM / SOM — 方法论与占位

## 1. 定义（须与财务/法务对齐口径）

| 层级 | 典型定义 | 本项目提示 |
|------|----------|------------|
| **TAM** | 全球可数字撮合的 **当地体验 / 向导类** 旅行服务支出（或 GMV 上限假设） | 引用第三方行业报告 + 链接存档 |
| **SAM** | TT 可合规开展业务的 **司法辖区 ∩ 支付能力** 子集 | 与 **`spec/84`** 十国叙事可对照，**勿把 84 直接等同 SAM 边界**（84 含融资地理，SAM 含合规） |
| **SOM** | 3～5 年 **可执行** 收入或 GMV 上限（保守 / 基础 / 乐观三档） | 与 **`09-business-model`** 中 GMV→Revenue 一致 |
