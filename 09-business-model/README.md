## 现金真实性校验（Real Money Check）

任何 **增长 / 实验 / 试点 / 验证**（含 **`10-growth-engine`**、**`15-data-decision`**、**`99-real-world-validation`**）在 **结案或扩量评审** 时须 **书面**回答（可挂在 PRD 附录、**`15/06`** 台账、**`99/00`** 证据目录）：

1. **是否产生真实收入**（非仅 GMV、非仅补贴扭曲的「好看数字」）  
2. **是否可回款**（现金或可动用结算；与 **权责发生** 区分）  
3. **是否改善现金流**（含营运资本、退款、资金占用、跑道）

**任一条为「否」且无已登记的风险承受 / 止损边界** → 标 **虚增长（Hollow Growth）** —— **不得**单独作为 **扩量、融资对外叙事** 的 **唯一** 支柱证据；须回 **`06-Cash-Conversion-Loop现金转化闭环.md`**、**`22-executive-dashboard/`**、**`21-product-operating-system/`** 决定是否停补贴、缩实验或升级 **`21/06`**。

**与 `99` 一致**：**`99/README`** 中的商业三问与本节 **同向**；冲突时以 **财务 / `spec/08-4` / 链上可审计路径** 为准。
