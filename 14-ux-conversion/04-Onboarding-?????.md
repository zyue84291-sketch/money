# Onboarding：注册 → 首单（魔法时刻）

## 1. 目标

缩短 **Time-to-first-Escrowed**（首单入金），且 **不牺牲 KYC/合规**。

## 2. 检查清单

| # | 项 |
|---|-----|
| 1 | 注册后 **下一步唯一 CTA** 是否明确（找向导 / 发布需求） |
| 2 | **空状态** 是否有示例内容与种子供给（`10-growth-engine`） |
| 3 | 首单前是否完成 **必要** 钱包教育（可跳过 vs 不可跳过写死） |
| 4 | 首单 **金额建议** 是否降低试错成本（产品策略，非 spec） |
